# Clusterを用いたメタバース村松城の開発と教育的有用性の検討

## 概要
本プロジェクトは、新潟県五泉市の村松城跡を対象に、メタバースを用いた歴史学習の可能性を検討したものです。

## 開発環境・使用技術
本プロジェクトの開発には、以下のプラットフォームおよびツールを使用しています。

メタバースプラットフォーム
・Cluster: 本ワールドの公開および実行環境 

制作ツール
・Unity
・Blender

分析ツール
jamovi (Version 2.6.45.0): アンケートデータの統計解析（主成分分析）
